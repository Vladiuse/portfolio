# Valentine Cards Telegram Bot

A Telegram-based application for sending Valentine cards inside a company.

# Essence

The project is a Telegram bot that allows users to create and send Valentine cards to other employees. The bot was available before February 14 and delivered all cards at a fixed time.

# Features

  • Select a predefined Valentine image or upload a custom one
  • Write a custom message
  • Choose a recipient from the list of employees
  • Send a Valentine either with the sender’s name or anonymously

# Delivery logic

All created Valentines were stored in the database and were not delivered immediately. On February 14 at 14:00, users could open the bot and view their inbox with all received Valentine cards.

# Technical details

  • Telegram Bot API
  • Step-by-step state-based user flow
  • Image upload and processing
  • Scheduled delivery logic
  • Access control for recipients

# Result

The bot was actively used by employees and successfully handled the full flow: from creating Valentine cards to scheduled delivery and viewing received messages.
