Work is check simple one page sites, there must:

  • Correct product price and discount
  • Correct product name
  • Set correct lang in admin

  • Correct dates and there format
  • No doubles of avatars in comments

How LinkChecker help?

  • Searching countrys on site, some times it dont must be there
  • Check is rekvisites added on page

It's being decided by load checked site in Iframe, add on page some CSS and JS code and text analize of site on backend.
