# Api order tester (work project)

GitHub: https://github.com/Vladiuse/-OI-AMK/

Its a python modul that can create orders, without using user interface. One of my job procces, i need to test, is orders from site send correctly. For it you must do a lot of job by hands:

  • Go to site
  • Find needed product
  • Create streem
  • Add some information in streem settings
  • Find in table phones numbers for current countrys
  • Send orders
  • go to tracker and find orders to see there status

My app just need to set product ID and chose countrys that you need to send orders.

All other work app will do it for you.
